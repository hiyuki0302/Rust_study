mapは「リストの各要素に同じ処理を適用して、新しいリストを作る」操作です。
// 数値変換
[1, 2, 3].map(|x| x * 2)     // [2, 4, 6]

// 型変換
[1, 2, 3].map(|x| x.to_string()) // ["1", "2", "3"]

// 複雑な処理
["hello", "world"].map(|s| s.len()) // [5, 5]

// イテレータ
let v1 = vec![1, 2, 3];
let iter = v1.iter().map(|x| x + 1);
let v2: Vec<i32> = iter.collect();  
→[2, 3, 4]
イテレータからあらたな配列を作成する。
